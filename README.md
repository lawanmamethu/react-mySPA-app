# React SPA - Single Page Application

![React](https://img.shields.io/badge/React-18.2.0-61DAFB)
![Status](https://img.shields.io/badge/Status-Completed-brightgreen)

## 📌 About

A modern Single Page Application (SPA) built with React. Features smooth navigation between Home, About, and Contact pages without reloads, global state management, and fully responsive design.

**Completed during Front-End Internship**

---

## ✨ Features

- 🏠 Home Page – Hero section, skills, featured projects, interactive counter
- 📘 About Page – Bio, work experience, education, technical skills
- 📧 Contact Page – Contact info + working form with validation
- ⚡ React Router for smooth navigation
- 🎨 Context API for state management
- 📱 Fully responsive design

---

## 🛠️ Tech Stack

- React 18
- React Router DOM
- Context API
- JavaScript (ES6)
- HTML5 & CSS3
- Git & GitHub

---

## 🚀 Quick Start

```bash
git clone https://github.com/lawanmamethu/react-mySPA-app.git
cd react-mySPA-app
npm install
npm start

```
